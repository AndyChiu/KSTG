# KSTG
鍵帽存放托盤產生器 Keycap Storage Tray Generator
版本: 1.0

此產生器使用OpenSCAD來打造存放鍵帽的托盤，可以有效運用廢棄盒子，依盒子尺寸與鍵帽尺寸來建立最合適的托盤，讓空間有效運用，並減少廢棄物。
